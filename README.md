## Challenge Tasks

### Task 1 — Fibonacci Generator

Create an infinite Fibonacci generator using `yield`.

### Task 2 — Running Minimum Generator

Yield the smallest value seen so far after each new element.

### Task 3 — itertools.islice()

Use `itertools.islice()` to print the first 20 Fibonacci numbers.

### Task 4 — Lazy Log Parser

Read a log file lazily and yield only lines containing:

```text
ERROR
```

### Task 5 — Memory Comparison

Compare memory usage between:

* Generator
* List

using:

```python
sys.getsizeof()
```
